Context Capsule API

Portable execution context for agent workflows.

Package the facts, current state, and next-step intent an agent needs to continue reliably.

Core value props:
 - less context window waste
 - better multi-agent handoffs
 - structured execution context (memory + state + intent)
 - fresh, shareable, action-ready, ephemeral
 - pairs with ProofSlip for verified agent workflows

Positioning:
 - ContextCapsule = navigational ("what's the situation and what to do next")
 - ProofSlip = evidential ("what actually happened, verified")
 - Together: infrastructure for reliable agent workflows
 - Umbrella brand: FUTURE (only after both have users)
